# 更新日志

英文主更新日志见 [CHANGELOG.md](CHANGELOG.md)。

## v4.2

- 缩短英文界面标签，避免小面板被英文长文本撑宽。
- 给英文短标签控件补充悬停提示。
- 英文和中文两个脚本版本号同步更新到 4.2。

## v4.1

- 项目改名为 `R34Video Watch Archive Downloader`。
- 中文显示名改为 `R34Video 观看归档下载+自动批量下载`。
- 维护英文和中文两个脚本版本。
- 英文脚本作为主版本，脚本内容不含中文界面文本。
- README 和 CHANGELOG 改为英文主文档，中文文档使用 `_ZH` 后缀。

## v4.0

- 在批量采集和下载流程基础上增加已看视频归档能力。
- 保留真实跳转、AJAX 页面、伪跳转和内嵌视频信号监听。

## v3.8

- 通过 `href` 和 `data` 属性读取分页页码，改进总页数统计。
- 保留 v3.7 的已采集页数显示。
- 当时作为正式版发布到 `r34video-bulk-downloader.user.js`。

## v3.7

- 在高级选项中增加 `已采集 x/y 页` 显示。
- 根据成功采集的页面更新已采集页数。

## v3.6

- 整合批量下载、自动下载、自动跳页、元信息保存、YT-DLP 导出、并行解析、子页面监听、设置同步、循环采集和下载统计。

## v3.5

- 增加循环采集：到最后一页后回到第一页，检测重复项目后停止。
- 顶部统计增加已提交和已下载计数。
- 统一面板按钮样式。

## v3.4

- 浏览器下载顺序改为先保存 `.meta.json`，再提交视频下载。
- 增加 `GM_addValueChangeListener`，支持跨标签页同步设置。
- 将队列持久化和设置同步拆分为两套存储。

## v3.3

- 增加 AJAX / 伪跳转视频页监听。
- 监听点击、DOM 变化、history 变化、前进后退、hash、pageshow 和 URL 轮询。
- 增加延迟检查，避免重复入队。

## v3.2

- 对持久化写入做防抖。
- 增加并行解析设置。
- 增加自动入队后自动下载当前视频。
- 缓存 UI 节点并减少重复解析。

## v3.1

- 初始扩展版：批量采集、自动翻页、视频直链解析、浏览器下载、直链导出、YT-DLP 命令导出、元信息 JSON 和已看视频自动入队。
